# makerthon
[실행 방법]
1. Python 3.9 이상 설치
2. 터미널(cmd 또는 PowerShell)에서 app.py가 있는 폴더로 이동
   cd 폴더경로
3. 필요한 패키지 설치
   python -m pip install -r requirements.txt
4. 실행
   python -m streamlit run app.py
5. 브라우저에서 자동으로 열리는 로컬 주소(예: http://localhost:8501)에서 확인

[카카오 REST API 키 발급 방법 - 실제 위치 검색 기능을 쓰려면 필요]
1. https://developers.kakao.com 접속 후 로그인 (카카오 계정, 무료)
2. 상단 메뉴 "내 애플리케이션" → "애플리케이션 추가하기"
3. 앱 이름 아무거나 입력 후 생성
4. 생성된 앱 클릭 → 왼쪽 메뉴 "앱 키" → "REST API 키" 복사
5. 왼쪽 메뉴 "카카오맵" → "사용 설정"을 ON으로 변경 (2024.12.1 이후 신규 앱은 이 설정이 꼭 필요합니다)
6. 앱 실행 후 사이드바 "카카오 REST API 키" 칸에 복사한 키를 붙여넣기

[API 키 없이도 사용 가능]
사이드바에 API 키를 입력하지 않으면, 메인 화면에서
"기본 이동시간 직접 입력/수정하기" 체크박스를 통해 수동으로 이동시간을 입력해
그대로 개인화 예측 기능을 사용할 수 있습니다.

[참고 - 거리 계산 방식]
무료 카카오 로컬 API에는 "도보 길찾기" 전용 API가 없어서,
검색된 두 좌표 사이의 직선거리를 구한 뒤 도로 사정을 반영하는 보정 계수(1.3배)를 곱하고
평균 도보속도(시속 4km)로 나누어 "지도앱 기본 예상시간"을 추정합니다.
실제 서비스로 발전시킨다면 이 부분을 티맵/네이버 도보 길찾기 API로 교체하면 더 정확해집니다.

[이동 이력]
travel_history.csv 파일로 저장되어 앱을 껐다 켜도 유지됩니다.
